This project is a minimal, multi-client, real-time chat server built using Node.js's standard `net` (socket) library, with no external frameworks or databases..

---

### 1. Prerequisites
You need **Node.js** installed on your system.

### 2. File Structure
Ensure your server code is saved as a single file: `server.js`.

### 3. Execution Command (SERVER SIDE)
Open your terminal in the project directory and run the server:

bash: to start the server
node server.js

### 4. Execution Command (CONNECTING THE CLIENT)

bash: to access the chat server
telnet 127.0.0.1 4000 

### 5. Execution Command (CLIENT SIDE)

bash: to login
LOGIN username

bash: to MSG
MSG text
